= RightnowOms

== Description

A common gem for managing your carts and orders.

== Installation

Add RightnowOms to your Gemfile.

  gem 'rightnow_oms'

== Development

RightnowOms is developed with Ruby 1.9.3-p0 and Rails 3.1.3

  bundle install
  bundle exec rake app:db:migrate
  rails s
